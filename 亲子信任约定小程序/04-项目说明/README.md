# 亲子信任约定小程序 — 项目说明

## 项目简介

"电子拉钩"是一款面向亲子关系的信任约定小程序原型。通过游戏化的方式，帮助孩子和家长建立、记录和维护彼此的承诺，在互动中培养守信意识，增进亲子信任。

## 原型使用方法

### 直接打开
双击 `03-原型代码/电子拉钩_亲子信任约定小程序.html` 即可在浏览器中运行。

> **注意**：建议使用本地服务器方式打开，以确保 localStorage 功能正常：
> ```bash
> cd "03-原型代码"
> python3 -m http.server 8080
> # 然后在浏览器访问 http://localhost:8080
> ```

### 首次使用流程
1. 首次打开会进入引导页，依次完成：
   - 选择喜欢的头像
   - 填写昵称
   - 添加常用约定人（如妈妈、爸爸）
2. 完成后自动进入首页
3. 后续打开将直接进入首页，不再重复引导

## 核心功能演示

### 1. 新建约定
- 点击首页底部「新建约定」入口栏
- 选择约定双方（发起承诺人 + 接受承诺人）
- 输入约定内容，AI自动解析
- 双方签名确认，生成约定卡片

### 2. 履约打卡
- 在首页「进行中的约定」列表中点击约定卡片
- 点击「已完成」按钮确认履约
- 系统弹出鼓励提示，信任树叶子变绿

### 3. 逾期复盘
- 逾期约定会显示黄色标识
- 点击「去复盘」按钮
- 填写原因，获得AI鼓励反馈
- 复盘后叶子恢复绿色

### 4. 信任树可视化
- 进入「信任树」页面查看整体进度
- 树上每片叶子代表一个约定
- 绿色 = 已兑现，黄色 = 待复盘
- 底部显示本周进度环形图

## 注意事项

1. **无痕模式**：在浏览器无痕/隐私模式下，localStorage 数据不会被持久化，刷新后会丢失
2. **文件协议**：直接双击 HTML 文件打开（file:// 协议）可能导致 localStorage 不稳定，建议使用本地服务器
3. **注销账户**：个人页面底部有「注销账户」按钮，点击后会清除所有数据并返回引导页
