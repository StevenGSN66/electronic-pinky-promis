# 信任星球 — 亲子信任约定小程序 产品规格说明

> 版本：v2.0  
> 日期：2026-07-15  
> 体验地址：https://stevengsn66.github.io/electronic-pinky-promis/electronic-pinky-promise.html

---

## 一、产品定位

**一句话定义**：一款面向亲子家庭的 AI 辅助信任约定工具，通过"电子拉钩"的仪式感 + 游戏化的信任树成长，帮助孩子和家长建立、记录和维护彼此的承诺。

**核心用户**：3-12 岁孩子的家长及孩子本人，以家庭为单位使用。

---

## 二、核心功能

### 2.1 信任约定全流程

| 阶段 | 功能 | 说明 |
|------|------|------|
| 创建 | 语音/文字输入约定 | 支持语音输入，AI 自动解析结构化约定 |
| 确认 | 电子拉钩签名 | 双方 Canvas 手写签名，生成约定凭证 |
| 履约 | 打卡确认 | 完成后双方确认，信任树获得水滴 |
| 逾期 | AI 复盘 | 雁小约引导双方填写原因，给出温和建议 |

### 2.2 信任树成长系统

- **7 个成长阶段**：种子 → 新芽 → 树苗 → 绿叶 → 开花 → 森林 → 星球
- **可视化展示**：树上每片叶子代表一个约定，绿色=已兑现，黄色=待复盘
- **守信值评分**：基于完成约定数计算家庭守信值

### 2.3 雁小约 AI 助手

- **全程语音/视频引导**：首页、录入页、AI 思考过程均有雁小约动画视频引导
- **智能解析**：DeepSeek AI 自动提取约定双方、截止时间、奖惩标准
- **温和复盘**：逾期后 AI 生成鼓励性建议，帮助家庭重新约定

### 2.4 首次引导

- 三步引导：选择头像 → 填写昵称 → 添加家庭成员
- 雁小约欢迎动画视频，带声音自动播放

## 三、技术规格

| 项目 | 说明 |
|------|------|
| 技术栈 | 纯 HTML + CSS + JavaScript（单文件） |
| 数据存储 | localStorage + sessionStorage |
| 响应式 | 移动端优先，桌面端侧边栏适配 |
| AI 集成 | DeepSeek API（前端直连演示版） |
| 语音输入 | Web Speech API |
| 部署 | GitHub Pages |
| 文件大小 | ~200KB（HTML）+ 资源文件 |

---

## 四、页面结构

| 页面 | 功能 |
|------|------|
| 引导页 | 首次进入：头像选择、昵称、家庭成员设置 |
| 首页（约定页） | 约定列表、AI 创建入口、空状态星球展示 |
| 创建约定页 | 双方选择、语音/文字输入、AI 解析、电子签名 |
| 信任树页 | 信任树可视化、守信值、本周进度 |
| 复盘页 | 逾期原因填写、AI 建议、信任树恢复 |
| 我的页 | 个人档案、勋章、账户管理 |

## 五、视觉设计

- **主色调**：薄荷绿 `#7CD4A8` / `#A8E6CF`
- **辅助色**：暖粉 `#FFB7B2`、暖黄 `#FFE66D`
- **风格**：清新、温暖、游戏化，适合亲子场景
- **图标**：自定义插画（雁小约吉祥物、信任星球、约定手势）
- **动画**：CSS 动画 + 视频引导（透明背景 WebM）
