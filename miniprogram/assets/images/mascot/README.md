# 雁小约 V2 吉祥物素材索引

> 用途：电子拉钩 / 信任星球 App Demo  
> 注意：当前素材尺寸可用于 Demo，但 PNG 暂无真正 Alpha 透明通道，正式版建议重新导出透明背景。

## 文件清单

| 文件 | 用途 |
| --- | --- |
| `yan-xiaoyue-main-front.png` | 首页主视觉、新人引导、品牌展示 |
| `mascot-front.png` | 通用站立形象、空状态、个人页 |
| `mascot-side.png` | 角色设定页、装扮页、设计规范 |
| `mascot-back.png` | 三视图展示、角色设定 |
| `mascot-welcome.png` | 新人欢迎、注册引导、首次进入 |
| `mascot-success.png` | 守约成功、完成打卡、获得水滴 |
| `mascot-reminder.png` | 到期提醒、温柔催办、订阅消息引导 |
| `mascot-thinking.png` | AI 复盘、AI 追问、思考状态 |

## 小程序引用示例

```xml
<image
  class="mascot"
  src="/assets/mascot/yan-xiaoyue-v2/mascot-welcome.png"
  mode="aspectFit"
/>
```

## 使用建议

```text
浅色背景优先。
不要放在深色复杂背景上。
同一屏尽量只出现一个雁小约主形象。
提醒、成功、复盘页面使用对应表情，不要混用。
```
