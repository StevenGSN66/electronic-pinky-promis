## 【正文】

### 1. Demo 简介

**是什么**：信任星球是一款面向亲子家庭的 AI 辅助信任约定 Web 应用，通过"电子拉钩"的仪式感 + 游戏化的信任树成长，帮助孩子和家长建立、记录和维护彼此的承诺。

**面向谁**：核心用户为 6–12 岁小学生及其家长，以家庭为单位使用。

**主要功能**：

| 功能 | 说明 | 截图建议 |
|------|------|---------|
| **信任约定全流程** | 语音/文字输入约定 → DeepSeek AI 自动解析结构化信息 → 双方 Canvas 电子签名 → 履约打卡 → 逾期 AI 复盘 | 截取创建约定页 + 签名页 |
| **信任树成长系统** | 完成约定获得信任水滴，树从种子成长为星球（7 个阶段），树上每片叶子代表一个约定，绿色=已兑现，黄色=待复盘 | 截取信任树页 |
| **雁小约 AI 助手** | 全程视频/语音引导（首页、录入页、AI 思考过程均有动画），逾期后 AI 生成温和复盘建议 | 截取首页雁小约引导 |

> 产品截图提示：请截取首页、创建约定页、信任树页各一张附在提交中。

### 2. Demo 创作思路

**灵感来源**：作为五年级小学生，平时大人随口许诺的约定经常找借口不算数，真的需要随时记录、监督和履行。如果有个小程序能随时记录、经常检查、到时候监督履行约定，应该能解决很多争吵。当然，这个约定也可以是双向的——孩子也有许诺给大人的时候，比如玩游戏的时间限制。

**想解决的问题**：

- **口头承诺无标准化留存**：家长、孩子均容易遗忘约定内容与截止时间，双方记忆分歧频发
- **约定落空后互相指责**：缺少中立温和的复盘渠道，激化亲子矛盾
- **传统工具缺乏仪式感与正向激励**：偏向监督管控，孩子难以建立守信成就感

**为什么做这个方向**：

亲子信任是一个被忽视但极其重要的家庭教育命题。市场上有很多习惯打卡、任务管理工具，但几乎没有专门为"亲子双向约定"设计的产品。信任星球的核心差异是"双向"——不只是家长给孩子布置任务，而是双方互相约定、共同守约，用"电子拉钩"的仪式感和游戏化的成长系统，让守信变得有成就感。

### 4. TRAE 实践过程

**开发流程**：

整个 Demo 从 0 到可用，完全通过 TRAE 对话式开发完成，核心流程如下：

| 步骤 | 内容 | TRAE 作用 |
|------|------|-----------|
| ① 基础骨架 | 生成包含 5 个核心页面的单文件 HTML 原型 | TRAE 一次性输出完整页面结构和导航逻辑 |
| ② 信任树可视化 | 实现 7 阶段成长树 + 叶子约定卡片覆盖 | TRAE 根据设计参考图实现 CSS + JS 渲染 |
| ③ 接入 DeepSeek AI | 编写 AI 解析 Prompt + API 调用逻辑 | TRAE 自动提取约定结构化信息（双方、时限、标准） |
| ④ 语音输入 | 集成 Web Speech API | TRAE 实现语音转文字 + 录音状态管理 |
| ⑤ 品牌视觉升级 | 替换雁小约吉祥物、信任星球 logo、约定图标 | TRAE 自动处理图片背景（Python PIL）、调整布局、统一色调 |
| ⑥ 视频引导 | 雁小约动画视频处理与播放控制 | TRAE 用 ffmpeg 去背景转 WebM，实现播放策略（一次播放 + 45秒空闲重播） |
| ⑦ 部署上线 | GitHub Pages 持续部署 | TRAE 自动完成 git 提交、gh-pages 分支合并与推送 |

### 5. 价值与意义（补充）

**效率提升价值**：

- AI 自动提取承诺人、时限、执行标准，替代手动手写记录
- 系统自动推送到期提醒，减少家长反复叮嘱的沟通内耗
- 标准化复盘流程降低家庭矛盾

**社会价值**：

- 以低门槛数字化工具落地守信家庭教育
- 摒弃说教式管教，用仪式化互动培养儿童责任意识
- 搭建温和平等的亲子沟通桥梁，助力家庭诚信氛围建设
